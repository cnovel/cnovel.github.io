Liste des projets auquel j'ai contribué

Modèles 3D

Ce mini-site regroupe les modèles 3D que j’ai réalisé avec ContextCapture. ContextCapture est un logiciel de photogrammétrie, c’est à dire qu’il est capable de retrouver la forme 3D des objets directement à partir de photos.

Secret Hitler en français

Secret Hitler est un jeu de société américain. Le jeu est disponible sous licence CC BY-SA-NC, c’est à dire qu’il est possible de le modifier tant que les auteurs originaux sont mentionnés et qu’aucune utilisation commerciale n’est effectuée. En me basant sur les fichiers originaux, j’ai entièrement redessiné les cartes à jouer et traduit entièrement le jeu.

Podcast Bulk Downloader

Podcast Bulk Downloader est un utilitaire Windows qui permet de télécharger tous les épisodes d’un flux podcast en quelques clics. Il existe une version ligne de commande et une version avec interface graphique. Le programme est codé en Python.

prenoms

prenoms est un package Python qui génère des prénoms et de noms aléatoirement depuis une liste ordonnée par fréquence. Le code sous licence MIT est disponible sur GitHub.

Pybimun

Pybimun est un utilitaire Windows qui permet de télécharger les images numérisées par les Bibliothèques Municipales Spécialisées de Paris en un clic à partir d’une URL. Le programme est codée en Python.
