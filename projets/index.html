## [Modèles 3D](http://xerto.free.fr/Acute3D "Lien vers les modèles 3D") Ce mini-site regroupe les modèles 3D que j'ai réalisé avec *ContextCapture*. *ContextCapture* est un logiciel de photogrammétrie, c'est à dire qu'il est capable de retrouver la forme 3D des objets directement à partir de photos.
![Mignature](/img/projets/sh-round.png)
## [Secret Hitler en français](/projets/secrethitler/ "Lien vers le site Secret Hitler") *Secret Hitler* est un jeu de société américain. Le jeu est disponible sous licence CC BY-SA-NC, c'est à dire qu'il est possible de le modifier tant que les auteurs originaux sont mentionnés et qu'aucune utilisation commerciale n'est effectuée. En me basant sur les fichiers originaux, j'ai entièrement redessiné les cartes à jouer et traduit entièrement le jeu.
![Mignature](/img/projets/pbd-round.png)
## [Podcast Bulk Downloader](https://github.com/cnovel/PodcastBulkDownloader "Lien vers le dépôt Github du projet") *Podcast Bulk Downloader* est un utilitaire Windows qui permet de télécharger tous les épidosdes d'un flux podcast en quelques clics. Il existe une version ligne de commande et une version avec interface graphique. Le programme est codé en Python.
